# Ace

## Overview
Ace is an offensive pilot focused on ability uptime. Reducing cooldowns allows mechs to use their abilities more frequently, which is especially valuable for high-burst mechs.

## Ability
**Quick Reflexes** — Reduces the mech's active ability cooldown by 25%.

## Best Mechs
- **Falcon** — More frequent Evades dramatically increases survivability and aggression
- **Mauler** — More frequent ability use enables sustained high-burst rotations
- **Ares** — More Overclock windows per match significantly increases kill potential

## Implant Synergies
- **Speed Booster** — Maximizes repositioning between ability uses
- **Damage Amplifier** — Each additional ability use deals more damage

## Tier
**S-tier** for ability-dependent burst mechs
